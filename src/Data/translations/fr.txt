ROMS_CHECKDAT_HELPDECIDE_BODY=<h3>MAME - dernière version</h3><p>Vous aurez besoin d'un fichier DAT correspondant à votre version de MAME : vérifier en utilisant un fichier DAT d'une autre version pourrait vous renvoyer beaucoup d'erreurs.</p><p>Si vous utilisez une distribution "tout en un" comme Batocera ou Recalbox, vous allez devoir trouver quelle version vous utilisez, car chacune intègre une version spécifique de MAME et FBNeo.</p><p><a href="https://wiki.batocera.org/arcade#romset_version_per_stable_batocera_release" class="blank">Table des versions de Batocera</a><br><a href="https://wiki.recalbox.com/fr/emulators/arcade/mame/libretro-mame" class="blank">Dernière version de Recalbox</a></p><p>Téléchargez ensuite le zip correspondant depuis <a href="https://www.progettosnaps.net/dats/MAME/" class="blank">ProgettoSnaps</a> et récupérez le fichier DAT dans le répertoire <em>DATs\ROMs</em>.</p><p>Si vous utilisez simplement MAME sur Windows, vous pouvez obtenir le XML en lançant <em class="code">.\mame.exe -listxml > mame-roms.xml</em> dans une ligne de commande.</p><h3>MAME 2003 Plus</h3><p>Mame 2003 Plus, utilisé dans les distributions qui ciblent le matériel peu puissant (Knulli, OnionOS, MuOS, mais aussi Batocera et Recalbox sur Raspberry Pi) n'est pas sur ProgettoSnaps, et pire, il en existe différentes versions/déclinaisons qui utilisent des romsets incompatibles.<br>Vous allez devoir extraire le XML depuis MAME:</p><ul><li>Lancez un jeu dans MAME</li><li>Ouvrez le menu RetroArch ; vérifiez le manuel pour savoir comment faire, mais c'est généralement une combinaison de boutons avec la hotkey</li><li>Ouvrez Quick menu > Core options > System > Activate MAME menu</li><li>Quittez le menu et continuez le jeu (retour plusieurs fois > quick menu > resume)</li><li>Allez en bas du menu noir qui apparaît au-dessus du jeu, et sélectionnez "Generate XML DAT"</li></ul><p>Si vous n'avez pas ces entrées de menu, il est possible que vous soyiez en train d'utiliser FBNeo.</p><p>The XML est généralement généré dans le répertoire "save" (ex : <em>CurrentProfile\MAME 2003-Plus\mame2003-plus</em> sur Miyoo Mini/OnionOS, ou <em>saves\mame2003-plus</em> sur Knulli) et appelé quelque chose comme <em>mame2003-plus.xml</em>.<br>Si vous ne le trouvez pas à cet endroit, eh bien, bonne chance.</p><h3>FBNeo</h3><p>Si vous utilisez FBNeo, utilisez simplement le fichier fourni, car les romsets ne changent généralement pas beaucoup d'une version à l'autre, sauf pour ajouter de nouveaux jeux.</p>
